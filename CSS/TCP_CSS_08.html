font-family

폰트 Times New Roman, Times, serif 를 사용한 문장입니다.
컴퓨터에 Times New Roman 폰트가 없으므로
이 문장은 Times가 사용되었습니다.

font-style

폰트 스타일을 italic을 사용한 문장입니다.
그리고 폰트는 serif를 사용하였습니다.

font-variant

폰트 variant를 Small-Caps 로 설정한 문장입니다.
대문자는 그대로, 소문자는 작은 폰트의 대문자로 변경되었습니다.

font-weight

폰트 weight가 lighter로 설정된 문장
폰트 weight가 bold로 설정된 문장
폰트 weight가 bolder로 설정된 문장
폰트 weight가 mine으로 설정된 문장

font-size

글꼴의 크기를 2.5em으로 변경한 문장 (배수단위)

글꼴의 크기를 20px로 변경한 문장 (픽셀단위)
